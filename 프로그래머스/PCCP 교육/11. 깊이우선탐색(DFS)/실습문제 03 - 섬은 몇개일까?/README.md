## 입력 사항

- 지도의 너비 W와 높이 H가 주어지고 (5 <= H <= 100, 5 <= W <= 100)
- 지도의 모양이 주어집니다.
- 1은 섬, 0은 바다입니다.
- 섬 끼리 서로 상 하 좌 우로 붙어있다면 그 섬들은 하나의 섬입니다.

<br>
